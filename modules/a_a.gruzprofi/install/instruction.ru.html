Инструкция по установке и настройке решения «ГрузПрофи»

1. Установка модуля

  1. Скачайте архив с решением из Маркетплейса.
  2. В административной панели Битрикс перейдите в раздел Marketplace → Установленные решения.
  3. Нажмите «Загрузить решение» и выберите скачанный архив.
  4. После загрузки нажмите «Установить».
  5. Следуйте шагам мастера установки:
    • Укажите название компании, телефон, email и часы работы.
    • Введите реквизиты (ИНН, ОГРН) для отображения в футере.
    • Укажите ссылку на Telegram для кнопки связи.
    • Введите API-ключ Яндекс.Карт (если есть).
  6. После завершения мастера сайт будет доступен по основному адресу.

2. Получение API-ключа Яндекс.Карт

  1. Перейдите в Кабинет разработчика Яндекса.
  2. Авторизуйтесь под своей учётной записью.
  3. Нажмите «Создать проект» и выберите тип «JavaScript API и HTTP Геокодер».
  4. В настройках ключа укажите домен вашего сайта.
  5. Скопируйте полученный ключ и вставьте его в соответствующее поле при установке модуля.
  6. Если ключ не был указан при установке, его можно добавить позже в файле /index.php в строке подключения API.

3. Настройка контента

3.1. Редактирование тарифов калькулятора

  1. Перейдите в Контент → Инфоблоки → Контент ГрузПрофи → Тарифы калькулятора.
  2. Отредактируйте существующие элементы или добавьте новые.
  3. Для каждого тарифа укажите:
    • Название — отображается в выпадающем списке калькулятора
    • Базовая цена — стоимость в рублях
    • Тип тарифа — B2C (для физлиц) или B2B (для юрлиц)

3.2. Редактирование автопарка

  1. Перейдите в Контент → Инфоблоки → Контент ГрузПрофи → Автопарк.
  2. Для каждой машины заполните:
    • Название
    • Грузоподъемность
    • Длина кузова
    • Объем кузова
    • Идеально для (описание)
    • Цена от (руб/час)
    • Хит заказа (Y/N)
    • Картинка анонса

3.3. Редактирование преимуществ

  1. Перейдите в Контент → Инфоблоки → Контент ГрузПрофи → Преимущества.
  2. Для каждого преимущества укажите название, описание и иконку (emoji).

3.4. Редактирование отзывов

  1. Перейдите в Контент → Инфоблоки → Контент ГрузПрофи → Отзывы.
  2. Заполните: имя клиента, текст отзыва, оценку (1-5), источник отзыва.

3.5. Редактирование контента главного экрана

  1. Перейдите в Контент → Инфоблоки → Контент ГрузПрофи → Контент хедера.
  2. Отредактируйте заголовок, подзаголовок и список преимуществ.

3.6. Редактирование B2B баннера

  1. Перейдите в Контент → Инфоблоки → Контент ГрузПрофи → B2B баннер.
  2. Настройте заголовок, текст, преимущества, ссылку на Telegram и текст кнопки.

4. Редактирование контактов через включаемые области

Все контактные данные хранятся в папке /local/include/ и редактируются прямо на сайте:

  1. Включите режим правки (кнопка «Изменить» в админ-панели).
  2. Наведите на нужный текст (телефон, часы работы, email).
  3. В открывшемся окне измените текст и сохраните.

Список редактируемых файлов:

  • phone.php — телефон
  • work_hours.php — часы работы
  • footer_company_name.php — название компании
  • footer_inn.php — ИНН
  • footer_ogrn.php — ОГРН
  • footer_email.php — email
  • telegram.php — кнопка Telegram
  • и другие

5. Удаление модуля

При удалении модуля автоматически удаляются:

  • Шаблон сайта /local/templates/gruzprofi/
  • Включаемые области /local/include/
  • Тип инфоблоков «Контент ГрузПрофи» и все созданные инфоблоки
  • Страница /privacy/

Главная страница /index.php заменяется на стандартную пустую страницу Битрикса. Если до установки модуля существовал другой index.php, он будет восстановлен из резервной копии.

6. Часто задаваемые вопросы

В: Карта Яндекс не отображается, что делать?
О: Проверьте, что вы указали API-ключ при установке или добавьте его вручную в файле /index.php в строке script src="https://api-maps.yandex.ru/2.1/?apikey=ВАШ_КЛЮЧ&lang=ru_RU".

В: Как изменить цены в калькуляторе?
О: Цены настраиваются в инфоблоке «Тарифы калькулятора». Измените значение свойства «Базовая цена».

В: Как добавить свою машину в автопарк?
О: В инфоблоке «Автопарк» нажмите «Добавить элемент» и заполните все поля.

В: Как изменить ссылку на Telegram?
О: Отредактируйте включаемую область /local/include/telegram.php через режим правки на сайте.
