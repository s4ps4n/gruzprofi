Политика конфиденциальности

Настоящая Политика конфиденциальности (далее — «Политика») определяет порядок обработки и защиты персональных данных пользователей, которые могут быть получены при использовании решения «ГрузПрофи».

1. Общие положения

1.1. Настоящая Политика разработана в соответствии с Федеральным законом от 27.07.2006 № 152-ФЗ «О персональных данных».

1.2. Решение «ГрузПрофи» не осуществляет самостоятельный сбор и обработку персональных данных конечных пользователей сайта. Все данные, вводимые на сайте, обрабатываются владельцем сайта самостоятельно.

1.3. Решение может содержать формы обратной связи, которые передают данные на сервер владельца сайта.

2. Какие данные могут обрабатываться

2.1. При использовании Решения владелец сайта может настраивать формы для сбора следующих данных:

  • Имя пользователя
  • Номер телефона
  • Адрес электронной почты
  • Другие данные, необходимые для оказания услуг грузоперевозок

2.2. Решение не передает собираемые данные третьим лицам, кроме случаев, предусмотренных законодательством.

3. Использование файлов cookie

3.1. Решение может использовать файлы cookie для обеспечения корректной работы сайта и улучшения пользовательского опыта.

3.2. Владелец сайта вправе настроить уведомление об использовании файлов cookie в соответствии с требованиями законодательства.

4. Интеграция со сторонними сервисами

4.1. Решение интегрируется со следующими сторонними сервисами:

  • Яндекс.Карты — для отображения карты и меток машин. При использовании карты Яндекс может собирать данные о местоположении пользователя в соответствии со своей политикой конфиденциальности.
  • Telegram — для связи через мессенджер. При переходе по ссылке Telegram может собирать данные в соответствии со своей политикой конфиденциальности.

4.2. Владелец сайта самостоятельно несет ответственность за информирование пользователей об интеграции со сторонними сервисами.

5. Защита данных

5.1. Решение не хранит персональные данные конечных пользователей в своей базе данных. Все данные, вводимые в формы, передаются владельцу сайта.

5.2. Владелец сайта самостоятельно обеспечивает защиту получаемых персональных данных в соответствии с законодательством РФ.

6. Ответственность

6.1. Разработчик Решения не несет ответственности за действия владельца сайта по сбору, обработке и хранению персональных данных пользователей.

6.2. Владелец сайта обязуется использовать Решение в соответствии с законодательством о защите персональных данных.

7. Изменения в политике конфиденциальности

7.1. Разработчик Решения оставляет за собой право вносить изменения в настоящую Политику. Новая редакция Политики вступает в силу с момента её публикации.

8. Контакты

8.1. По всем вопросам, связанным с настоящей Политикой, обращайтесь через Маркетплейс «1С-Битрикс».
