Лицензионное соглашение

Настоящее Лицензионное соглашение (далее — «Соглашение») заключается между Вами (далее — «Пользователь») и разработчиком решения «ГрузПрофи» (далее — «Правообладатель»).

1. Предмет соглашения

1.1. Правообладатель предоставляет Пользователю право использования программного обеспечения «ГрузПрофи» (далее — «Решение») на условиях простой (неисключительной) лицензии.

1.2. Решение предназначено для создания сайта грузоперевозок на платформе «1С-Битрикс: Управление сайтом».

1.3. Пользователь вправе использовать Решение на одном экземпляре (одной установке) платформы «1С-Битрикс».

2. Права и ограничения

2.1. Пользователь вправе:

  • Устанавливать и использовать Решение в соответствии с его функциональным назначением.
  • Модифицировать Решение для собственных нужд.
  • Создавать резервные копии Решения.

2.2. Пользователь не вправе:

  • Распространять Решение или его модифицированные версии без письменного разрешения Правообладателя.
  • Передавать права использования Решения третьим лицам.
  • Удалять или изменять информацию об авторских правах, содержащуюся в Решении.

3. Техническая поддержка

3.1. Правообладатель предоставляет техническую поддержку Решения через Маркетплейс «1С-Битрикс».

3.2. Срок технической поддержки составляет 1 (один) год с момента приобретения Решения.

3.3. Техническая поддержка включает:

  • Консультации по установке и настройке Решения.
  • Исправление ошибок в работе Решения.
  • Предоставление обновлений Решения.

4. Гарантии и ответственность

4.1. Решение предоставляется «как есть». Правообладатель не гарантирует отсутствие ошибок в Решении.

4.2. Правообладатель не несет ответственности за любой ущерб, связанный с использованием или невозможностью использования Решения.

4.3. Правообладатель гарантирует, что Решение не содержит вредоносного кода.

5. Срок действия соглашения

5.1. Настоящее Соглашение вступает в силу с момента установки Решения и действует бессрочно.

5.2. Соглашение может быть расторгнуто в случае нарушения Пользователем условий настоящего Соглашения.

6. Заключительные положения

6.1. Настоящее Соглашение регулируется законодательством Российской Федерации.

6.2. Все споры, возникающие из настоящего Соглашения, подлежат разрешению в соответствии с законодательством РФ.
